util.js

Contains common utility functions.

Functions Description
toQueryString

toQueryString(queryObject)

Argument(s):

  • queryObject: Object

Return Value: String

Takes an object value and returns a formatted string which can be transformed back into an object using the toQueryObject function.

A single character prefix is added to the values to represent initial type.
toQueryObject

toQueryObject(queryString)

Argument(s):

  • queryString: String

Return Value: Object

Takes a formatted string, returned by toQueryString function, and returns an object
escapeODataValue

escapeODataValue(str)

Argument(s):

  • str: String

Return Value: String

Used in oData ($filter) query string to remove special characters not allowed by the oData service.
swapView

swapView(element, newView[, oldView])

Argument(s):

  • element: HTMLElement
  • newView: Backbone.View
  • oldView: Backbone.View

Return Value: Backbone.View

Transitions from one (oldView) Backbone View to another (newView) Backbone View located inside (element) an HTMLElement, returning the new (newView) Backbone View.

In addition to swapping between two Backbone Views, it freezes the element's height to prevent flickering between two sizes.
ajax

ajax(options)

Argument(s):

  • options: Object

Return Value: Promise

A wrapper for the Jquery ajax function.
adjustArgsadjustArgs

adjustArgs(signature,...args)

Argument(s):

  • signature: Array
  • ...args: Any

Return Value: Array

Takes an array of type of values (signature) and match it to a list or arguments (...args) returning an array of values in the order of the signature information.
